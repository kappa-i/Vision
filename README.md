## À quoi sert Vision

Quand on travaille avec un client, le feedback se disperse vite : mails,
WhatsApp, WeTransfer, appels… On perd du temps, le client se sent peu impliqué,
et la direction artistique se dilue.

Vision réunit tout au même endroit : la direction artistique, l'avancement, les
retours, la validation et la livraison. Vous créez l'espace du projet, vous
invitez votre client, et chacun voit la même chose, en temps réel.

Ce n'est pas un outil de gestion de tâches, ni un simple transfert de fichiers :
c'est l'espace de la **relation créatif ↔ client**.

## Installer l'application

Téléchargez la dernière version depuis la page **Releases**.

### Windows
Lancez le fichier `.exe` (ou `.msi`) et suivez l'installation.

### macOS
1. Téléchargez le fichier `.dmg` (il fonctionne sur tous les Mac).
2. Ouvrez-le et glissez **vision** dans le dossier **Applications**.
3. Au premier lancement, macOS peut afficher *« vision est endommagé »*. C'est
   normal (l'app n'est pas encore signée par Apple), ce n'est pas un virus.
   Pour débloquer une bonne fois : ouvrez l'app **Terminal** et collez cette
   ligne, puis Entrée :
   ```
   xattr -d com.apple.quarantine /Applications/vision.app
   ```
4. Ouvrez l'application depuis le dossier Applications.

## Démarrage rapide

1. **Créez votre compte** (email + mot de passe) au premier lancement.
2. **Créatif** : créez un projet, puis cliquez sur **Inviter le client** pour
   obtenir un code.
3. **Client** : sur l'écran d'accueil, cliquez sur **Rejoindre via un code** et
   saisissez le code reçu.
4. Vous travaillez maintenant dans le même espace, synchronisé en temps réel.

## Côté créatif

Vous êtes propriétaire du projet. Un espace projet s'organise en onglets :

- **À propos** — le tableau de bord du projet : où on en est, la prochaine
  action à mener, les chiffres clés. C'est aussi ici qu'on définit l'**image de
  couverture** du projet.
- **Moodboard** — posez votre direction artistique : images de référence,
  rangées en dossiers. Glissez-déposez vos images directement.
- **Galerie** — téléversez vos photos (bouton ou glisser-déposer), rangez-les
  par albums, et proposez une **comparaison avant / après** au curseur.
- **Livraison** — choisissez les photos approuvées à livrer, puis cliquez sur
  **Remettre au client** quand tout est prêt. Tant que vous ne l'avez pas fait,
  le client ne peut pas télécharger.

Pour donner accès à votre client : bouton **Inviter le client** → un code est
généré (valable 72 h, à usage unique). Transmettez-le-lui.

## Côté client

Vous rejoignez le projet de votre créatif avec un code. Vous pouvez :

- **Valider photo par photo** — sur chaque visuel, **Approuver** ou
  **Demander une révision**. Vos décisions font avancer le statut du projet.
- **Annoter** — cliquez sur une image pour épingler un commentaire à un endroit
  précis (« éclaircir ici », « recadrer »…). Les retours réglés peuvent être
  marqués comme résolus.
- **Donner un feedback global** — l'onglet **Feedback** est un fil de
  discussion pour le projet dans son ensemble.
- **Télécharger** — une fois le projet remis par le créatif, récupérez vos
  fichiers (un par un, ou **Tout télécharger**).

Vous pouvez aussi **quitter un projet** que vous avez rejoint : il disparaît de
votre espace, sans rien supprimer chez le créatif.

## Le déroulé d'un projet

1. Le créatif **crée le projet** et pose sa direction artistique dans le
   **Moodboard**.
2. Il **invite le client** avec un code.
3. Le client rejoint, découvre la direction, et réagit.
4. Le créatif **téléverse l'avancement** dans la **Galerie**.
5. Le client **annote, commente, approuve** ou demande des révisions.
6. Le créatif ajuste, puis **choisit les fichiers à livrer**.
7. Il **remet le projet** → le client **télécharge** ses fichiers finaux.

À chaque étape, des **pastilles** signalent les nouveautés (sur le tableau de
bord et sur les onglets), et de petites **notifications** confirment vos
actions.

## Questions fréquentes

**Le client doit-il installer l'application ?**
Oui, pour l'instant Vision est une application à installer (Windows ou macOS).
Une version web pour les clients, sans installation, est prévue.

**Mes photos sont-elles protégées ?**
Le clic droit et le glisser des images sont désactivés, et l'application est
exclue des captures d'écran (totalement sur Windows ; les screenshots sont
bloqués sur macOS, mais pas l'enregistrement vidéo, à cause d'une limite du
système). Une protection plus poussée (aperçus en basse définition avec
filigrane avant la livraison) est prévue.

**Si je réinstalle l'app, est-ce que je perds mes projets ?**
Non. Vos projets, photos et retours sont enregistrés en ligne. Réinstaller
l'application ne touche pas à vos données.

**Le client peut-il supprimer mon projet ?**
Non. Un client peut seulement **quitter** un projet (le retirer de son espace).
Seul le créatif, propriétaire du projet, peut le supprimer.

**Comment mettre à jour l'application ?**
Téléchargez la nouvelle version depuis la page Releases et réinstallez-la (sur
macOS, relancez la commande de déblocage). La mise à jour automatique viendra
plus tard.
